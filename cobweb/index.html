Lyapunov spectrum  λ(r) across the whole map

Where λ dips below 0 (green) the orbit is periodic; where it pokes above 0 (red) it's chaotic. The downward spikes inside the red are periodic windows — islands of order. Space play/pause · click the curve to set r.
