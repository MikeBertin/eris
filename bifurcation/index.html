The Road to Chaos

Iterate one line of arithmetic — x → x² − r — and ask where it settles. For small r it lands on a single value; raise r and that value splits in two, then four, then eight, in a period-doubling cascade that piles up into chaos. Drag a box to zoom — the tree is a fractal, the same forks all the way down. Ported from my undergrad computational-physics coursework.
